
## Uruchom Codespaces

Uruchom Codespaces korzystając z `Code` -> `Codespaces` -> `New codespace` (przycisk `+`).

![](imgs/run-codespaces.png)

## Po starcie

```bash
az login
sudo az aks install-cli
```
